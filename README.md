ℹ️ Информация о данном моде:
- Бинды: R - TriggerBot, Z - ESP
- Статус модуля выводится в action bar

💡 ESP позволяет видеть игроков через стены в виде подсветки (как от спекральнвх стрел)
⚔️ (1.21.4) TriggerBot позволяет бить сущностей: дистанция - 3 блока, без сброса спринта, бьет через стены (может не работать на популярных серверах)
⚔️ (1.21.11) TriggerBot не рекомендуется к использованию, дистанция 3-4 блока, криво бьет через стены, бьет через щит (ANTICHEAT DETECT❗)

⏲️ Установка

1. Установи [Fabric Loader](https://fabricmc.net/use/)
2. Скачай последнюю версию мода из [Releases](https://github.com/O3HUX/ESP-TriggerBot-Minecraft/release)
3. Перемести скачанный `.jar` в папку `.minecraft/mods`
4. Запускай игру
